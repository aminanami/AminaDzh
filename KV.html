Контрольные Вопросы

Что такое «система контроля версий»?

Система контроля версий (СКВ) — это инструмент, который помогает отслеживать изменения в файлах и управлять версиями этих файлов. Она позволяет нескольким пользователям работать над одним проектом, сохраняя историю изменений и обеспечивая возможность отката к предыдущим версиям.

К какому типу VSC относится? В чем главное отличие от других типов?

Git относится к распределенным системам контроля версий. Главное отличие от централизованных систем заключается в том, что в Git каждый разработчик имеет полную копию репозитория на своем компьютере, что позволяет работать в оффлайне и снижает риски потери данных.

В чем заключается преимущество использования Git?

Преимущества использования Git включают возможность работы в оффлайне, высокая скорость операций, возможность создания веток для параллельной разработки, а также мощные инструменты для слияния изменений и разрешения конфликтов.

Какие команды Git вы знаете?

Некоторые основные команды Git: git init (инициализация репозитория), git add (добавление изменений в индекс), git commit (сохранение изменений), git push (отправка изменений на удаленный репозиторий), git pull (получение изменений из удаленного репозитория).

В чем отличие Git и GitHub?

Git — это система контроля версий, которая позволяет отслеживать изменения в коде. GitHub — это веб-сервис, который использует Git для размещения репозиториев и совместной работы над проектами, предоставляя дополнительные инструменты для управления проектами, отслеживания ошибок и взаимодействия с сообществом разработчиков.
